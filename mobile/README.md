Shaxe Mobile (Expo)

Quick start:

1. Install Expo CLI if you don't have it:

```bash
npm install -g expo-cli
```

2. Install dependencies and run:

```bash
cd mobile
npm install
npm start
```

3. Use Expo Go on your phone or an emulator to preview.

Notes:
- Backend API base URL defaults to `http://localhost:5000/api`. Use `10.0.2.2` for Android emulator if needed.
- This scaffold includes basic screens and components; integrate with backend endpoints next.
